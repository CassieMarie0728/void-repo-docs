# Android Build Guide

The complete, current Android guide is maintained at [`docs/android.md`](docs/android.md).

Quick commands:

```powershell
npm run android:apk
npm run android:aab
```

Read the full guide for toolchain discovery, Capacitor synchronization, remote API configuration,
device testing, signing, release versioning, and black-screen diagnostics. Skipping those sections
and jumping directly to store upload is certainly a choice, just not a particularly survivable one.
